# CSS

Selektor

```css
p {
  font-size: 12pt;
}
```

- `*` - označi vse
- `#`... - označi id
- `.`... - označi class

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>RIC Poklicna matura</title>
  </head>
  <body>
    <ul>
      <li>RIC</li>
      <li>MATURA</li>
      <li>POKLICNA MATURA</li>
      <li>VPRAŠANJA</li>
    </ul>
    <div class="crta" />
  </body>
</html>
```

```css
.crta {
  background-color: #1a5276;
  height: 3px;
  width: 100%;
}
```
